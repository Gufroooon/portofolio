About

Halo! Saya Gufron Wicaksono, seorang siswa jurusan Rekayasa Perangkat Lunak (RPL). Sebagai seorang pelajar yang fokus dalam pengembangan aplikasi dan sistem perangkat lunak, saya telah mempelajari berbagai keterampilan yang memungkinkan saya untuk mengembangkan solusi digital yang efektif dan efisien. Selama belajar di jurusan RPL, saya tidak hanya mendapatkan pengetahuan teori, tetapi juga berkesempatan untuk langsung mengaplikasikan ilmu yang saya pelajari melalui berbagai proyek dan tugas praktikum. Saya memiliki keahlian dalam pengembangan perangkat lunak menggunakan berbagai bahasa pemrograman, mulai dari HTML, CSS, PHP, Dan JavaScript. Selain itu, saya juga mengembangkan kemampuan dalam desain antarmuka pengguna (UI/UX), memastikan bahwa aplikasi yang saya buat tidak hanya berfungsi dengan baik, tetapi juga mudah digunakan dan menyenangkan bagi penggunanya. Di luar pengembangan aplikasi, saya juga tertarik pada web development dan pengelolaan database untuk menciptakan sistem yang stabil dan aman. Saya telah membuat banyak web yang saya masukkan di portfolio saya ini, mulai dari HTML CSS, Web Bootstrap, Web Design menggunakan figma, Pembuatan Website memakai WordPress dan lain lain. Sebagai Siswa Jurusan RPL, Saya juga memiliki banyak pengalaman seperti Uji Sertikom yang saya tampilkan di portofolio.
